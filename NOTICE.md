### What Digital Dark Adds

- Always-on dark mode UI theme with custom Metal L&F
- Vibrant 16-color per-component rendering
- Spotlight component search (F2)
- Keyboard shortcuts editor with persistence
- Configurable component font size (6–48pt)
- Signal-state-aware pin dot coloring
- Shift-click multi-select for components and wires
- Tooltips with keyboard shortcut display
- Modern flat toolbar styling
